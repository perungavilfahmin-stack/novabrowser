# MyBrowser

MyBrowser is a lightweight Python-based browser project designed as a learning and experimental tool for understanding how web navigation, shortcuts, and browser-like interfaces work.

## 🚀 Features

- Basic web navigation (back, forward, refresh)
- URL/search input support
- Keyboard shortcuts for faster control
- Lightweight and fast Python implementation
- Simple and clean interface
- Custom project-based browser logic
